# ignite-rentalx

# About

This is a car rental backend application.

The project's requirements are in the `Requirements.md` file.

## Usage

Install dependencies with

```bash
npm install
```

Start the containers by running

```bash
docker-compose up -d
```

Run the application

```bash
npm run dev
```
